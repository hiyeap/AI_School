부트스트랩이란?

웹 프론트개발에서의 구조를 미리 만들어준 프레임워크

각종 버튼, 테이블, 입력창 등 여러가지의 css style과 레이아웃 정렬을 class만으로 제어할 수 있다.

시작하기

테마를 직접 설치하지 않고 URL을 통해 불러오는 CDN 방식을 이용하기

head 내부에 link태그, script태그를 이용하여 css, javascript를 사용할 수 있도록 한다.

그리드 레이아웃

상위 요소에 container가 필요하다.

col-md-숫자를 사용하기 위해 row 클래스를 지정한다.

col-md-숫자에서 숫자의 합은 12가 되어야한다.

모바일 : class="col-xs-숫자"
태블릿 : class="col-sm-숫자"
데스크톱 : class="col-md-숫자"
더 큰 데스크톱 : class="col-lg-숫자"
